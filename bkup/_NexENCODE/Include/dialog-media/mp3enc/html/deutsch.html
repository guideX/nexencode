Software MPEG Audio layer 3 Abspieler.

 Hoch komprimierte Audiodateien in hoher Qualität.
Beispiel in VB 5.0 !

1) Fügen Sie die OCX in VB unter PROJEKT/KOMPONENTEN ein und in dem Komponenten Fenster erscheint ein Icon mit eine CD & Zauberstab.

2) Für das erste Programm verwenden Sie bitte einen COMMAND Element mit dem folgenden Programm

Private Sub Command1_Click()
Dim err As Long
 err = Mp3Enc1.Open("d:\test.wav", "d:\test.mp3")
 ' "d:\test.wav" ist der Pfad der Wave Datei ( bitte anpassen )
' "d:\test.mp3" ist der Pfad der MP3 Datei ( bitte anpassen )
Mp3Enc1.Encode
End Sub

3) Das war es schon. Jetzt können Sie MPEG Audio in Ihrem Programm benutzen.

4) Die OCX im arbeitet mit allen Windows 95 und NT 4 Versionen. Die ActiveX benötigt zusätzlich ein Update ( mfc 4.02 ).
