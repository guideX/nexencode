Software only encoder for MPEG Audio layer 3.

 High compression audio files with high quality sound.

How to use in VB 5.0 !

1)Insert OCX in VB under PROJECT/COMPONETS and a cd with a magic wand icon apears in the components window

2) For the first step you can use a COMMAND button with the following two lines

Private Sub Command1_Click()
Dim err As Long
 err = Mp3Enc1.Open("d:\test.wav", "d:\test.mp3")
 ' "d:\test.wav" is the path of the wave file
 ' "d:\test.mp3" is the path of the wave file
 Mp3Enc1.Encode
End Sub

3) Thats all ! Now you can use MPEG Audio layer 3 in your program.

4) The OCX in the root work with all Windows 95 and NT 4 version. For the ActiveX version you need a Windows update ( mfc 4.02 ).
