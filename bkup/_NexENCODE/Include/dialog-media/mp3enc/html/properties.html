Properties ( all Read-only otherwise noted ):

AllowDownsample: True or False; use 22kHz instead of 44kHz ( faster )

BitRate Bitrate of MP3-stream
( in KHz: 32, 40,48,56,64,80,96,112,128,160,192,224,256,320 )

FrameCount: Number of frames in MP3-stream

FrameNotifyCount( RW ): Sent notify event every x frames ( default 32 )

IsCopyrighted: Return info from MPEG stream

IsOriginal: Return info from MPEG stream

HasChecksums: Return info from MPEG stream

SampleRate: Use the information from a valid wave riff header. If  no valid header then ( 32000, 44100, 48000 ) is possible.

IsCopyrighted: Set the copyright Bit.

IsOriginal: Set the original Bit

Version: Return version of OCX version * 100
