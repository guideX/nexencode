This zip file contains the OCX and the ActiveX version of mp3enc.ocx. The OCX version is in the root directory and the ActiveX is in the ActiveX directory.

The OCX need no additional files to work with Windows 95 or NT 4. The ActiveX needs the MFC 4.2 files to work correct.

You can use the zip files without any changes on a CD or website. Please take care to use the new version from our website !
