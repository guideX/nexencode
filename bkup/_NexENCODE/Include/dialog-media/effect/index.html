How to use in VB 5.0 !

1) Insert OCX in VB under PROJECT / COMPONENTS and a the Effect-Icon apears in the components window

2) For the first step you can use a COMMAND button with the following two lines

Private Sub Command1_Click()
Dim err As Long
 err = Effect1.InputFileOpen("c:\song.wav")
 ' "c:\song.wav" is the path of the wav file
Effect1.Play
End Sub

3) Use another COMMAND button with the following text

Private Sub Command2_Click()
 Effect1.Echo (900, 90)
End Sub

4) Use another COMMAND button with the following text

Private Sub Command3_Click()
 Effect1.play
End Sub

5) The ActiveX / OCX work with all Windows 95 / 98 and NT 4 version.
